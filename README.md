## Setting Up

### Install

* Node.js (v16.15.1 or higher)
* NPM (v8.11.0 or higher)

### Then

```
$ npm install
$ npm run deploy # run only on production
$ npm start
```

## Start for development

```
$ npm run start-dev
```

## Build client app

```
$ npm run build
```
